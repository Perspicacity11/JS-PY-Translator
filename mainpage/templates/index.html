Editor's note: This app is powered by ChatGPT's publicly available API, through which I am using the DaVinci engine to generate formatted code snippets. As with any LLM, GPT does not actually 'know' how to translate code, but more how to properly return the most reasonable, likely answer based on its dataset (part of which is Common Crawl, ergo it has digested all code documentation up to 2021). As such, please recognise that this system will not in all cases return perfectly correct code. It is designed to translate snippets of around 250 words or similar elements. You have been warned.

Credit is due to Mosh for his excellent materials on learning Python fundamentals, and to Pyplane for his equally excellent tutorial on using the OpenAI plugin to build a chatbot. I cannot recommend their work highly enough.
